#                                       Web development / Web 开发

##                          1. 基础认知

###         1.1) Web 标准
* Web 标准中分成3个构成:

    结构                HTML                页面元素和内容
    表现                CSS                 网页元素的外观和位置等页面样式(如: 颜色, 大小等)
    行为                JavaScript          网页模型的定义与页面交互

###         HTML 认知
* HTML(Hyper Text Markup Language): 超文本语言
